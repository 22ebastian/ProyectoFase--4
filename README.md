# Sistema Integral de Gestión - Software FJ
## Curso: Programación (213023) - Ingeniería de Sistemas

### Descripción del Proyecto
Este sistema ha sido desarrollado para la empresa **Software FJ** con el fin de gestionar de manera robusta y estable sus servicios de:
* Reserva de salas.
* Alquiler de equipos.
* Asesorías especializadas.

### Características Técnicas
El sistema implementa de forma rigurosa:
* **POO Avanzada:** Abstracción, herencia, polimorfismo y encapsulación [2, 3].
* **Estabilidad Total:** Manejo de excepciones personalizadas y encadenamiento de errores para garantizar que la aplicación nunca se detenga [4, 5].
* **Auditoría:** Registro de todos los eventos y errores en un archivo de logs externo [6, 7].

### Estructura del Repositorio
* `Main.py`: Punto de entrada del sistema.
* `Simulaciones.py`: Ejecución de las 10 pruebas obligatorias (casos válidos e inválidos) [6, 7].
* `Cliente.py`, `Servicios.py`, `Reserva.py`: Lógica de negocio.
* `Excepciones.py`: Definición de errores personalizados.
* `Logger.py`: Gestión del archivo de registro de eventos.
* `Validador.py`: Validación estricta de parámetros [3, 8].

### Instrucciones de Ejecución
1. Clonar el repositorio.
2. Ejecutar el archivo `Main.py` para iniciar las simulaciones automáticas.
3. Revisar el archivo `archivo_de_logs.txt` para ver el historial de operaciones.
